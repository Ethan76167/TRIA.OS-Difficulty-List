**OTHER INFO** --> Skills*** | Awards* | Medal**

**VICTORS** --> Victors (usernames separated by commas)

Create a combination of numbers, where each number represents the skills used in the map.

2: Dive
3: Linear Sliding
4: Momentum Sliding
5: Orbs
